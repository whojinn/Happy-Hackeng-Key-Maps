# Happy-Hackeng-Key-Maps
AutoHotKeyでHHKBをさらにいい感じにしてみた。

# 機能要約
- `Ctrl+c -> BackSpace`
- `Ctrl+d -> Delete`
- `Ctrl+p -> UpAllow`
- `Ctrl+n -> DownAllow`
- `Ctrl+f -> RightAllow`
- `Ctrl+b -> LeftAllow`
- `Alt+c -> Ctrl+BackSpace`
- `Alt+d -> Ctrl+Delete`
- `Alt+p -> Ctrl+UpAllow`
- `Alt+n -> Ctrl+DownAllow`
- `Alt+f -> Ctrl+RightAllow`
- `Alt+b -> Ctrl+LeftAllow`
- `Ctrl+Shift+p -> Ctrl+p`
- `Ctrl+Shift+n -> Ctrl+n`
- `Ctrl+Shift+f -> Ctrl+f`
- `Ctrl+Shift+b -> Ctrl+b`
